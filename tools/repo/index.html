Hobby ARM OS Package Repository

This is the Docker-served repo the on-device package manager talks to. The HTML page you are reading was rendered by the built-in browser running on a hand-rolled AArch64 kernel through plain HTTP/1.0 -- no TLS, no JavaScript, no images.

Open-source packages

  • hello — one-shot greeting
  • counter — prints 1..5
  • clock — tick-tock demo
  • files — ram-fs lister
  • sysinfo — cpu/memory dump
  • load — scheduler exerciser

Games

  • minesweeper — 10x10, mouse-driven
  • sudoku — click + type

Closed-source samples

  • weather — proprietary stub

To install a package from inside the OS: pkg install <name> from the terminal, or click an icon in the App Store window.
